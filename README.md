## ts-selenium-starter

这是一个`Selenium + TypeScript + ESLint + Prettier`的起手式

这个工程的搭建笔记可以在这里[查看](https://welives.github.io/blog/front-end/automation/selenium-webdriver.html)

## 使用

```sh
pnpm install
```

## 开发模式

```sh
pnpm dev
```

### 打包

```sh
pnpm build
```
